My Research


Fire and Smoke Digital Twin (Present)

Underneath a $150,000 grant from the Austin Fire Department, I am developing a digital twin of the Austin area to forecast smoke falloff
and improve urban management. I am publishing two papers on this.

My Projects


Atmosphere Cloud (Present)

A peer-to-peer streaming platform that allows idle resources to be used for streaming custom VMs. This is a work in progress using C and Ruby, and is currently in the alpha phase, potentially available for startup funding.

Hallowed Island (October 2021)

Made as a one-week challenge for Halloween, this 3D first person shooter game features three uniquely scripted bosses, a huge environment, and an hour experience through a dungeon. This was developed primarily in C# and Unity.

Otaku Puzzle (July 2021)

Made as a response to a friend's challenge about "easy to make puzzle games", this spin on classic 2D puzzles games is 3D, top-down, includes full networking support through WebSockets, implements a search engine media scraper, and creates puzzles that support videos, GIFs, and most formats.

Update Launcher (July 2021)

Made in a few hours with Visual Studio after realizing Otaku Puzzle needed a system that fetched the most recent update, and applied it. This was fairly simple, and includes a tutorial on using it for your own apps. I utilized .NET Core and various Windows APIs to access registry keys for versioning and the file system.
